# Changelog

All notable changes to the Nova AI Resume Screener project will be documented in this file.

## [2.4.0] - 2026-04-17
### Added
- **Multi-Job Context:** Support for selecting specific Job Requisitions to score candidates against.
- **Bi-Axial Radar Analytics:** Comparative skill mapping for deep candidate analysis.
- **Command Palette:** Integrated Cmd+K support for rapid navigation.
- **Improved Rationale Engine:** Dynamically generated AI reasoning based on job-specific constraints.

### Fixed
- **Routing:** Implemented `404.html` fallback and `trailingSlash` configuration for reliable GitHub Pages path resolution.
- **Aesthetic:** Corrected Z-index issues with the ambient cursor spotlight on mobile devices.

## [2.0.0] - 2026-04-12
### Added
- **Awwwards-Level Polish:** Framer Motion orchestration, page transitions, and glassmorphic UI system.
- **Persistence:** LocalStorage hydration for candidate data.

## [1.0.0] - 2026-04-05
- Initial MVP release with basic resume parsing and dashboard.
